There are three files in this project, namely Janus-yaogan.py,train2(1).py, and inference2(2).py. Among them, Janus-yaogan.py is the code for processing images using the Janus-Pro-7B model and conducting habitability assessment. train2(1).py is the code for fine-tuning the training model, and inference2(2).py is the code for calling the DeepSeek model to process POI information for habitability assessment
Janus-yaogan.py takes in POI information and image data. The specific prompt should be designed based on the input information and evaluation criteria. The final output result will be selected from the designed evaluation options, which is more conducive to subsequent processing.
The difference between inference2(2).py and others lies in that it does not require the input of image data; instead, it directly inputs POI and AOI information to obtain the assessment results.
